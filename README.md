# 🤖 TGBots — Коллекция Telegram-ботов

Три Telegram-бота на Python, реализованных через прямые HTTP-запросы к Telegram Bot API — без использования сторонних библиотек.

---

## Стек технологий

| Компонент | Технология |
|-----------|-----------|
| Язык | Python 3 |
| Интеграция | Telegram Bot API (прямые HTTP-запросы через `requests`) |
| Внешние API | Погода, курсы валют, авиабилеты |
| Данные | JSON |

---

## Боты

### 🌤 Бот погоды
Получение актуальной погоды по названию города через внешний weather API.

**Команды:**
- `/weather [город]` — текущая погода в указанном городе

### 💱 Конвертер валют
Конвертация валют по актуальному курсу через внешний currency API.

**Команды:**
- `/convert [сумма] [из] [в]` — конвертация (например: `/convert 100 USD RUB`)

### ✈️ Поиск авиабилетов
Поиск авиабилетов по направлению через внешний API.

**Команды:**
- `/flights [откуда] [куда]` — поиск доступных рейсов

---

## Особенность реализации

Все боты работают напрямую с Telegram Bot API через HTTP-запросы (`requests.get` / `requests.post`) без использования библиотек-обёрток (aiogram, python-telegram-bot). Это даёт полное понимание протокола работы с Bot API.
